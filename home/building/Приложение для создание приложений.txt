

О приложение


Это облачная среда разработки, развернутая на моем домене. Пользователь получает доступ через клиентское приложение, которое после авторизации подключается к рабочей среде.

Ядро приложения — визуальный редактор по принципу Figma. Ты собираешь интерфейс из готовых блоков, и система сразу генерирует код в нужном формате под выбранный технологический стек. Дальше выбираешь задачу из списка, указываешь целевые устройства, и среда делает сборку готового решения с превью. Все операции чтения и записи связок происходят локально на ЭВМ пользователя. Приложение выступает ассистентом между пользователем и файловой системой.

Доступ к среде есть только у оплативших пользователей. В Supabase хранятся только данные об аккаунте и статусе доступа. Сами данные проектов и связки не уходят в облако — они лежат локально у пользователя.

Дополнительный слой — маркетплейс наработок через Gumroad. Каталог готовых связок подтягивается по API: обложка, название, цена. Проверка оплаты проходит через Gumroad API. Платежи, чекауты и выдача файлов полностью на стороне Gumroad. По клику пользователь переходит на Gumroad для покупки, а после оплаты импортирует связку в свою локальную IDE.

Итог: это визуальная IDE на моем домене с автогенерацией кода и локальным хранением. Supabase отвечает только за доступ. Интеграция с Gumroad нужна для обмена и монетизации готовых решений.
